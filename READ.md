# KIRA - KRIBHCO Intelligent Retrieval Assistant

KIRA is an AI-powered enterprise knowledge assistant built for KRIBHCO.

## Features

- RAG-based document retrieval
- FastAPI backend
- Local LLM integration (Ollama)
- PDF knowledge base
- Source citations
- Analytics dashboard
- Persistent question tracking
- KIRA branding

## Tech Stack

- Python
- FastAPI
- LangChain
- FAISS
- Ollama
- HTML/CSS/JavaScript
